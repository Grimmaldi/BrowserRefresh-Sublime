============================
Browser Refresh Update
============================

IMPORTANT BREAKING CHANGES

Hi there and thanks for using Browser Refresh. In this update I tried to
fix a couple of issues people where reporing. Mainly the wrong browser
being started when trying to refresh.

** WHAT TO DO **
Go to your settings file and make sure your set the browsers you want
to refresh when your press the keyboard shortcut. The default is Chrome.

Go to:
"Preferences" > "Package Settings" > "Browser Refresh" > "Key Bindings - User"

** DEFAULTS **
[
  {
    "keys": ["super+shift+r"], "command": "browser_refresh", "args": {
      "auto_save": true,
      "delay": 0.0,
      "activate": true,
      "browsers" : ["chrome"]
    }
  }
]
